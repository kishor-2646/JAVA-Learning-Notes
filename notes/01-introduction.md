# 01 — Introduction to Java

## What is Java?
- High-level, **object-oriented** programming language
- Developed by **Sun Microsystems** in 1995
- Invented by Canadian scientist **James Gosling** in 1991 under Sun Microsystems
- Originally named **"OAK"** (1991), renamed to **"Java"** in 1995 (copyright)
- "JAVA" is sometimes jokingly called "Just Another Virtual Application/Architecture" (backronym)
- Java was owned by Sun Microsystem, now under **Oracle**

## Use Cases
- Desktop applications
- Web applications
- Android apps
- Enterprise systems

## Features of Java

| Feature | Description |
|---------|-------------|
| OOP | Supports object-oriented programming to create modular, reusable code |
| Robust & Secure | Strong memory management & exception handling |
| Rich API & Standard Libraries | Extensive built-in libraries for various needs |
| Open-Source Libraries | Wide range to extend functionality and speed up development |
| Platform Independence | Write Once, Run Anywhere (WORA) — runs on any OS with JVM |
| Multithreading & Concurrency | Allows concurrent execution of multiple tasks |
| Frameworks for Enterprise & Web | Spring, Spring Boot, Struts, Hibernate |
| Maintainability & Scalability | Structured design allows easy maintenance |

## Famous Applications Built Using Java
Netflix, Amazon, LinkedIn, Minecraft, Spotify, Uber, NASA World Wind

## What Can We Do With Java?
- Mobile App Development (Android)
- Web Development (Spring, Spring Boot, Struts, Hibernate)
- Desktop GUI Applications (JavaFX, Swing)
- Enterprise Applications (banking, etc.)
- Game Development (LibGDX, jMonkeyEngine)
- Big Data Technologies (Hadoop, Apache Kafka)
- Cloud-based Applications (AWS, Azure, GCP)
